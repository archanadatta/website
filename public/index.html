Projects

Content-Based Movie Recommendation System

This project develops a content-based movie recommendation system using plot overviews from The Movie Database (TMDB). Built with Python and Streamlit, the system applies natural language processing techniques, specifically TF-IDF vectorization, to transform movie plots into numerical representations. Cosine similarity is then used to compute narrative closeness and suggest thematically similar films. The system emphasizes modularity, caching for performance, and a user-friendly interface that displays recommendations with metadata and posters. While challenges include sparse overviews, cold-start issues, and limited diversity, the project demonstrates a practical, lightweight solution and provides a foundation for future hybrid and deep learning enhancements.

Unified GAN-Based Framework for Multi-Task Learning

The project presents a unified GAN-based framework for diverse image transformation tasks including text-to-image generation, image-to-image translation, super-resolution, inpainting, and style transfer. By leveraging pre-trained models such as Stable Diffusion, CycleGAN, Pix2Pix, and ESRGAN, the system achieves high-quality outputs while reducing computational demands. An intuitive Streamlit interface integrates these capabilities, enabling users to upload inputs and view results in real time. The framework emphasizes efficiency, accessibility, and educational value, making advanced image processing feasible on modest hardware. It establishes a scalable foundation for future research, creative applications, and practical deployment in fields like design, restoration, and medical imaging.
